## Описание проекта  

Проект посвящён базовым навыкам администрирования Linux-систем. В процессе работы были выполнены следующие задачи:  

- Установка и настройка **Ubuntu Server** без графического интерфейса  
- Создание и управление пользователями, настройка прав через **sudo**  
- Настройка сети: статические и динамические IP, шлюз, DNS, проверка соединений  
- Обновление системы и управление пакетами  
- Настройка службы синхронизации времени  
- Работа с текстовыми редакторами (**vim**, **nano**, **mcedit** и др.)  
- Установка и настройка **SSH** (включая смену порта и автостарт)  
- Использование системных утилит (**top**, **htop**, **fdisk**, **df**, **du**, **ncdu**)  
- Анализ системных журналов (**/var/log/**)  
- Автоматизация задач с помощью планировщика **cron**  

Проект охватывает ключевые аспекты администрирования: от начальной установки и обновлений системы до работы с сетевыми сервисами, мониторинга ресурсов и настройки автоматизации.


# Задания

### Задание 1

**Установить Ubuntu 20.04 Server LTS без графического интерфейса.**
- Граф. интерфейс должен отсутствовать
- С помощью команды `cat/etc/issue` узнать версию убунту, приложить скриншот в отчет

### Задание 2

**Создать пользователя, добавить в группу adm**
- Вставить скриншот команды создания пользователя
- Вставить скриншот вывода cat/etc/passwd где видно нового пользователя

### Задание 3

- Задать машине имя user-1
- Установить временную зону соответствующую текущему местоположению
- Вывести сетевые интерфейсы с помощью консольной команды
    - Объяснить что такое lo в отчете
- Получить с помощью консоли ip адрес от DHCP-сервера
  - В отчете написать что такое DHCP
- Определить и вывести внешний и внутренний адрес IP-шлюза
- Задать статичные (заданные вручную) настройки ip, gw, dns (Использовать публичные DNS-серверы. Например 1.1.1.1 или 8.8.8.8)
- Перезагрузить машину, убедиться что настройки из предыдущего пункта остались

В отчете описать как были выполнены пункты.
